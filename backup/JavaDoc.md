# JavaDoc

- ==***`javadoc`命令***==：
  - 用来生成自己***API文档***的

- ***API文档***：
  - 看***index.html***

- 命令行：

```bash
#格式
#javadoc 参数 Java文件
javadoc -encoding UTF-8 -charset UTF-8 xxx.java
```

## 参数信息

- 文档注释：

```java
/**
 * @author 作者名
 * @version 版本号
 * @since 指明需要最早使用的jdk版本
 * @param 参数名
 * @return 返回值情况
 * @throws 异常抛出情况
 */
```
