# Java基础知识

## Java特性和优势

- 简单性
- 面向对象
  - 万物皆对象
- 可移植性
  - 跨平台
- 高性能
  - 即时编译
- 分布式
- 动态性
  - 反射
- 多线程
- 安全性
  - 捕获异常
- 健壮性



## Java三大版本

- `JVM`
  - Write Once、Run Anywhere



1. `JavaSE`：标准版（桌面程序，控制台开发......）
2. JavaME：嵌入式开发（手机，小家电......）
3. `JavaEE`：E企业级开发（web端，服务器开发......）



## JDK、JRE、JVM

- `JDK`：Java Development Kit
- `JRE`：Java Runtime Environment
- `JVM`：Java Virtual Machine
