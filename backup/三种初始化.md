# 三种初始化

- 静态初始化
  - ***创建 + 赋值***


```java
int[] a = {1,2,3};
Man[] mans = {new Man(1,1),new Man(2,2)};
```

- 动态初始化

```java
int[] a = new int[2];
a[0]=1;
a[1]=2;
```

- 数组的默认初始化
  - 数组是引用类型，它的元素相当于类的实例变量
  - 因此数组一经分配空间，其中的每个元素也被按照实例变量同样的方式被**隐式初始化**
- **隐式初始化**
  - **一般默认值是`0`**
