# 二维数组

- 多维数组可以看成是数组的数组
  - 比如**二维数组**
    - 就是一个特殊的一维数组
    - **其每一个元素都是一个一维数组**
- 二维数组
  - 解析：以下二维数组`a`可以看成一个两行五列的数组

```java
int a[][] = new int[2][5];
```

- 思考：多维数组的使用？

```java
num[1][0];
```
