# 包机制

- 包机制用于**区别类名的命名空间**
- 语法格式为：

```java
package pkg1[.pkg2[.pkg3...]];
```

- ==***一般用公司域名倒置作为包名***==

- 为了能够使用某一个包的成员
  - ***使用`"import"`语句导入该包***


```java
import pkg1[.pkg2...].(classname|*);
```
