# 可变参数

- `JDK1.5`开始，`Java`支持传递同类型的可变参数给一个方法
- 格式：
  - 在方法声明中，==***指定参数类型后加一个省略号（...）***==

- **一个方法只能指定一个可变参数，它必须是方法的最后一个参数。任何普通的参数必须在它之前声明**

```java
方法名(参数类型...参数名){
    //方法体
}
```
