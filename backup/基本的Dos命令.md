# 基本的Dos命令

## 打开CMD的方式

1. 开始 + 系统 + 命令提示符
2. `Win键+R`   输入`cmd`打开控制台（==推荐使用==）
3. 在任意的文件夹下面，按住`shift键 + 鼠标右键`点击，在此处打开命令行窗口（==推荐==）
4. 资源管理器的的地址栏前面加上 `cmd 路径`



管理员方式运行：选择以管理员方式运行



## 常用的Dos命令

```bash
#盘符切换
cd /d 路径
#查看当前目录下的所有文件 dir
#切换目录	cd	(change directory)
cd ..
#清理屏幕	cls	(clear screen)
#退出终端	exit
#查看电脑的ip	ipconfig

#打开应用
	calc
	mspaint
	notepad
	
#ping 命令
	ping www.baidu.com
	
#文件操作
	md	目录名
	rd	目录名
	cd>	文件名
	del	文件名
```
