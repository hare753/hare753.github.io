# 安装开发环境

## 卸载JDK

1. 删除Java的安装目录
2. 删除`JAVA_HOME`
3. 删除`path`下关于Java的目录
4. `java -version`



## 安装JDK

1. 浏览器搜索JDK8，找到下载地址
2. 同意协议
3. 下载电脑对应的版本
4. 双击安装`JDK`
5. **记住安装的路径**
6. 配置环境变量
   1. 我的电脑-->右键-->属性
   2. 环境变量-->`JAVA_HOME`
   3. 配置`path`变量
7. 测试`JDK`是否安装成功
   1. 打开`cmd`
   2. `java -version`



## JDK8下的环境

- bin（可执行文件）
  - java.exe
  - javac.exe
- include（C语言的一些头文件）
- jre（java运行环境）
- lib（java的一些库文件）
- src.zip（资源文件）
