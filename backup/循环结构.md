## `while`循环

- `while`是最基本的循环，它的结构为：

```java
while(布尔表达式){
    //循环内容
}
```

- 区别：

| 循环       | 方式               | 举例                 |
| ---------- | ------------------ | -------------------- |
| ***停止*** | ***表达式失效***   |                      |
| 一直执行   | 布尔表达式为`true` | 服务器的请求响应监听 |

## `do...while`循环

- 结构：

```java
do {
    //代码语句
}while(布尔表达式);
```

<img width="616" height="583" alt="Image" src="https://github.com/user-attachments/assets/fc3bd236-1fa9-41a1-9d53-bff9f3a00e04" />



- `While`和`do...While`的区别：

| 循环         | 执行顺序    | 主要差别                         | 先决条件                       |
| ------------ | ----------- | -------------------------------- | ------------------------------ |
| `while`      | 判断——>执行 |                                  | 不满足条件，则不能~~进入循环~~ |
| `do...while` | 执行——>判断 | ***保证循环体会被至少执行一次*** | 即使不满足条件，也至少执行一次 |

## ==`For`循环==

- `for`循环语句
  - 是支持迭代的一种通用结构，***是最有效、最灵活的循环结构***
- `for`循环执行的次数是在执行前就确定的
- 格式：

```java
for(初始化;布尔表达式;更新){
    //代码语句
}
```

## 增强`for`循环

### ==`JDK5`新特性==

- 这里我们先只是见一面，做个了解
  - 数组我们重点使用
- `Java5`引入
  - 一种主要用于数组或集合的
    - ***增强型`for`循环***
- 格式：

```java
for(声明语句 : 表达式)
{
	//代码句子    
}
```

- 声明语句：
  - 声明新的局部变量
    - 该变量的类型必须和数组元素的类型匹配
    - 其作用域限定在循环语句块
    - 其值与此时数组元素的值相等
- 表达式：
  - 表达式是要访问的数组名，或者是返回值为数组的方法
