# 注释

- 注释并不会被执行，是给我们写代码的人看的
- ==***书写注释***==是一个非常好的习惯 ***==BAT==***
- 平时写代码一定要==***注意规范***==



## Java中的注释有三种：

- 单行注释
- 多行注释
- 文档注释

```java
//单行注释	//
//多行注释	/*	注释	*/
//文档注释	/**	*/
```
