# 类型转换

- 运算中，不同类型的数据先转化为同一类型，然后进行运算

<img width="672" height="152" alt="Image" src="https://github.com/user-attachments/assets/b16d359f-47a3-4e95-b6cf-0f369ec48bda" />





| 类型     | 方式           | 方向   |
| -------- | -------------- | ------ |
| 强制转换 | （类型）变量名 | 高-->低 |
| 自动转换 | 无             | 低-->高 |



<img width="634" height="417" alt="Image" src="https://github.com/user-attachments/assets/276ee73b-0193-4062-bc51-239c692f22d3" />





## 注意点

1. 不能对布尔值进行转换
2. 不能把对象类型转换为不相干的类型
3. 在把高容量转换到低容量的时候，强制转换
4. 转换的时候可能存在==***内存溢出***==，或者==***精度问题***==!
