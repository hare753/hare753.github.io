# 运算符

- `Java`语言支持如下运算符：
  - 基础：
    - 算术运算符：+，-，*，/，%，++，--
    - 赋值运算符：=
    - 关系运算符：>，<，>=，<=，==，!= instanceof
    - 逻辑运算符：&&，||，!

  - **了解**：
    - 位运算符：&，|，^，~，>>，<<，>>>

  - 必须掌握：
    - 条件运算符：？：
    - 扩展赋值运算符：+=，-=，*=，/=

### 逻辑运算符

- 短路运算
  - 第一个条件为`false`，后面的都不用看了，直接判断为`false`
  - ***反之（`true`）则继续判断（按顺序判断）***

| 符号 | 名称          | 运算                              |
| ---- | ------------- | --------------------------------- |
| `&&` | 逻辑与（and） | `true&&true-->true`               |
| \|\| | 逻辑或（or）  | true\|\|false-->true 			  |
| `!`  | 非（取反）    | `!true-->false and !false-->true` |
