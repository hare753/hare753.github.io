# ==递归==

- A方法调用B方法，我们很容易理解！
- 递归：
  - A方法调用A方法！
  - 就是==***自己调用自己***==
- 递归的能力：
  - 用有限的语句来定义对象的无线集合

- **递归结构包括两个部分：**
  - **递归头：**
    - **什么时候不调用自身方法。如果没有头，将陷入死循环**
  - **递归体：**
    - **什么时候需要调用自身方法**
